
# Triangles

## Описание

Проект представляет из себя реализацию матрицы. Программа считает определитель. На вход подаётся размер n квадратной матрицы, а дальше n*n элементов матрицы построчно. На выходе значение определителя.

Вход:
```
2 1 0 0 1
```
Выход:
```
1
```

## Сборка

Зависимости проекта (c++20):

```bash
sudo apt install -y \
    cmake \
    build-essential \
    libgtest-dev \
```

Сборка проекта осуществляется следующими командами: 

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Release
cmake --build build
```

Запуск:
```bash
./build/matrix
```

## Отладка

Чтобы скомпилировать программы в режиме отладки, следует скомпилировать в режиме `Debug`

```bash
cmake -S . -B build -DCMAKE_BUILD_TYPE=Debug
cmake --build build
```
